# Microcosm

Agent-simulation platform for the built world — landing site.

- `index.html` — main landing page
- `demo.html` — interactive live demo (brief → seed → deliberation → report)
- `research.html` — research foundation + founder's memo (position, model, team, capital plan)
- `support.js` — generated design-runtime (do not edit)

Serve statically, e.g. `python3 -m http.server 4599`.
