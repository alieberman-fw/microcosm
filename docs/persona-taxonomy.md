# Built-World Persona Taxonomy — Master Library

Starter library for agent-swarm generation. ~1,100 base personas. Part I is the horizontal built-world library (37 categories by occupation/role). Part II is a set of asset-type build stacks — the specific expert rosters needed to take a given product type (data center, high-rise condo, mall, etc.) from dirt to operations. Part III is the technology & product-validation library — engineering, AI, product, GTM, leadership, and buyer-side personas for validating software ideas (and proptech ideas, by combining Part III with Parts I–II). Stacks list only asset-specific specialists; they compose with Part I for the generic roles (GC, civil engineer, lender, etc.). Each base persona instantiates against the modifier axes at the end of the doc, so the effective library is in the tens of thousands.
